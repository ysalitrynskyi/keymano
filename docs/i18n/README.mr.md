# Keymano — macOS कीबोर्ड लेआउट संपादक

> 🌐 हे भाषांतरित सारांश आहे. संपूर्ण दस्तऐवजीकरण इंग्रजीमध्ये आहे: **[README](../../README.md)**.

> 🟢 **GitHub वर नवीन किंवा डेव्हलपर नाही?** सोप्या इंग्रजीत **[प्रारंभ मार्गदर्शक](../../docs/GETTING_STARTED.md)** वाचा.

**Keymano** हे macOS `.keylayout` फाइल्स आणि `.bundle` कीबोर्ड पॅकेजेससाठी एक मोफत, मुक्त-स्रोत, क्रॉस-प्लॅटफॉर्म संपादक आहे — Ukelele ला एक मुक्त-स्रोत पर्याय. macOS, Windows, Linux, किंवा थेट तुमच्या ब्राउझरमध्ये, लाइव्ह, क्लिक करण्यायोग्य कीबोर्डसह macOS कीबोर्ड लेआउट तयार करा, संपादित करा, आणि तपासा.

## ते चालवण्याचे तीन मार्ग

- **डेस्कटॉप अॅप डाउनलोड करा** — macOS, Windows, किंवा Linux साठी, [Releases page](https://github.com/ysalitrynskyi/keymano/releases) वरून.
- **ते तुमच्या ब्राउझरमध्ये वापरा** — होस्ट केलेले अॅप **[keymano.ys.contact](https://keymano.ys.contact)** येथे उघडा (देखभालकर्त्याद्वारे होस्ट केलेले); पूर्ण अॅप तुमच्या ब्राउझरमध्ये चालते, कोणतीही स्थापना नाही.
- **स्रोतापासून बिल्ड करा** — योगदानकर्त्यांसाठी; [English README](../../README.md#build-from-source) पहा.

## वैशिष्ट्ये

- दृश्य संपादक — कोणत्याही मॉडिफायर संयोजन आणि डेड-की स्थितीसाठी त्याचे आउटपुट सेट करण्यासाठी कोणत्याही कीवर क्लिक करा
- डेड कीज, टर्मिनेटर्स, आणि मॉडिफायर मॅप्स
- ANSI / ISO / JIS भौतिक कीबोर्ड भूमिती
- Apple चे मूळ `.keylayout` XML आणि `.bundle` पॅकेजेस वाचते आणि लिहिते
- एक-क्लिक स्वयंचलित दुरुस्तीसह पडताळणी; PNG आणि संदर्भ-शीट निर्यात
- 24 भाषांमधील इंटरफेस

- **जतन करा vs. म्हणून जतन करा** — जतन करा सध्याची फाइल ओव्हरराइट करते; म्हणून जतन करा प्रत तयार करते.
## गोपनीयता

**डेस्कटॉप अॅप** काहीही गोळा करत नाही आणि ऑफलाइन चालते. **होस्ट केलेले वेब अॅप** [keymano.ys.contact](https://keymano.ys.contact) वर पर्यायी **Google Analytics** (अनामिक पृष्ठ-दृश्य आकडेवारी) वापरते. तुमचे `.keylayout` काम ब्राउझर सोडत नाही. सेल्फ-होस्टवर ऑपरेटर सक्षम करेपर्यंत विश्लेषण नाही. [PRIVACY.md](../../PRIVACY.md).
