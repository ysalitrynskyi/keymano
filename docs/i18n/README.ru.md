# Keymano — Редактор раскладок клавиатуры для macOS

> 🌐 Это переведённое краткое описание. Полная документация на английском: **[README](../../README.md)**.

> 🟢 **Новичок на GitHub или не разработчик?** Прочитайте простое **[руководство для начала (на английском)](../../docs/GETTING_STARTED.md)**.

**Keymano** — это бесплатный, с открытым исходным кодом, кроссплатформенный редактор для файлов macOS `.keylayout` и пакетов клавиатур `.bundle` — альтернатива Ukelele с открытым исходным кодом. Создавайте, редактируйте и проверяйте раскладки клавиатуры macOS с помощью живой, кликабельной клавиатуры на macOS, Windows, Linux или прямо в браузере.

## Три способа запустить его

- **Скачать настольное приложение** — для macOS, Windows или Linux, со [страницы Releases](https://github.com/ysalitrynskyi/keymano/releases).
- **Использовать в браузере** — откройте размещённое приложение на **[keymano.ys.contact](https://keymano.ys.contact)** (размещено сопровождающим); полное приложение работает в браузере, установка не требуется.
- **Собрать из исходного кода** — для участников проекта; см. [английский README](../../README.md#build-from-source).

## Возможности

- Визуальный редактор — нажмите любую клавишу, чтобы задать её вывод для любой комбинации модификаторов и состояния мёртвой клавиши
- Мёртвые клавиши, терминаторы и карты модификаторов
- Физические раскладки клавиатуры ANSI / ISO / JIS
- Чтение и запись нативных пакетов Apple `.keylayout` XML и `.bundle`
- Проверка с автоматическим исправлением в один клик; экспорт PNG и листов-справок
- Интерфейс на 24 языках

- **Сохранить vs. Сохранить как** — Сохранить перезаписывает текущий файл; Сохранить как создаёт копию.
## Конфиденциальность

**Настольное приложение** ничего не собирает и работает офлайн. **Размещённая веб-версия** на [keymano.ys.contact](https://keymano.ys.contact) по желанию использует **Google Analytics** (анонимная статистика просмотров). Ваша работа с `.keylayout` никогда не покидает браузер. Самостоятельный хостинг без аналитики, пока оператор её не включит. См. [PRIVACY.md](../../PRIVACY.md).
