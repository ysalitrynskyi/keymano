# Keymano — Edytor układów klawiatury dla macOS

> 🌐 To jest przetłumaczone streszczenie. Pełna dokumentacja jest po angielsku: **[README](../../README.md)**.

**Keymano** to darmowy, open-source’owy, wieloplatformowy edytor plików `.keylayout` dla macOS oraz pakietów klawiatury `.bundle` — otwartoźródłowa alternatywa dla Ukelele. Twórz, edytuj i sprawdzaj układy klawiatury macOS za pomocą interaktywnej, klikalnej klawiatury, na macOS, Windows, Linux albo bezpośrednio w przeglądarce.

## Trzy sposoby uruchomienia

- **Pobierz aplikację desktopową** — dla macOS, Windows lub Linux, ze strony [Releases page](https://github.com/ysalitrynskyi/keymano/releases).
- **Użyj w przeglądarce** — otwórz hostowaną aplikację pod adresem **[keymano.ys.contact](https://keymano.ys.contact)** (hostowaną przez opiekuna projektu); pełna aplikacja działa w przeglądarce, bez instalacji.
- **Zbuduj ze źródeł** — dla współtwórców; zobacz [English README](../../README.md#build-from-source).

## Funkcje

- Edytor wizualny — kliknij dowolny klawisz, aby ustawić jego wynik dla dowolnej kombinacji modyfikatorów i stanu martwego klawisza
- Martwe klawisze, terminatory i mapy modyfikatorów
- Fizyczne geometrie klawiatur ANSI / ISO / JIS
- Odczytuje i zapisuje natywne dla Apple pliki XML `.keylayout` oraz pakiety `.bundle`
- Walidacja z automatyczną naprawą jednym kliknięciem; eksport PNG i arkusza referencyjnego
- Interfejs w 24 językach
