# Keymano — macOS Tastaturlayout-Editor

> 🌐 Dies ist eine übersetzte Zusammenfassung. Die vollständige Dokumentation ist auf Englisch: **[README](../../README.md)**.

**Keymano** ist ein kostenloser, quelloffener, plattformübergreifender Editor für macOS `.keylayout`-Dateien und `.bundle`-Tastaturpakete — eine Open-Source-Alternative zu Ukelele. Erstelle, bearbeite und prüfe macOS-Tastaturlayouts mit einer live anklickbaren Tastatur, auf macOS, Windows, Linux oder direkt in deinem Browser.

## Drei Möglichkeiten, es auszuführen

- **Lade die Desktop-App herunter** — für macOS, Windows oder Linux, von der [Releases page](https://github.com/ysalitrynskyi/keymano/releases).
- **Verwende es in deinem Browser** — öffne die gehostete App unter **[keymano.ys.contact](https://keymano.ys.contact)** (gehostet vom Maintainer); die vollständige App läuft in deinem Browser, keine Installation erforderlich.
- **Aus dem Quellcode bauen** — für Mitwirkende; siehe die [English README](../../README.md#build-from-source).

## Funktionen

- Visueller Editor — klicke auf eine beliebige Taste, um ihre Ausgabe für jede Kombination aus Modifikatoren und Dead-Key-Zustand festzulegen
- Dead Keys, Terminatoren und Modifizierkarten
- Physische Tastaturgeometrien für ANSI / ISO / JIS
- Liest und schreibt Apples natives `.keylayout`-XML und `.bundle`-Pakete
- Validierung mit automatischer Reparatur per Klick; PNG- und Referenzblatt-Export
- Oberfläche in 24 Sprachen
