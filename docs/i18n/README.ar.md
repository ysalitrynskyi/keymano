# Keymano — محرر تخطيط لوحة مفاتيح macOS

> 🌐 هذا ملخص مترجم. التوثيق الكامل باللغة الإنجليزية: **[README](../../README.md)**.

> 🟢 **جديد على GitHub أو لست مطوّرًا؟** اقرأ **[دليل البدء (بالإنجليزية)](../../docs/GETTING_STARTED.md)** بلغة بسيطة.

**Keymano** هو محرر مجاني ومفتوح المصدر ومتعدد المنصات لملفات macOS `.keylayout` وحِزم لوحات المفاتيح `.bundle` — بديل مفتوح المصدر لـ Ukelele. أنشئ وحرّر وافحص تخطيطات لوحة مفاتيح macOS باستخدام لوحة مفاتيح تفاعلية قابلة للنقر مباشرة، على macOS وWindows وLinux، أو مباشرةً في متصفحك.

## ثلاث طرق لتشغيله

- **تنزيل تطبيق سطح المكتب** — لـ macOS أو Windows أو Linux، من [صفحة الإصدارات](https://github.com/ysalitrynskyi/keymano/releases).
- **استخدامه في متصفحك** — افتح التطبيق المستضاف على **[keymano.ys.contact](https://keymano.ys.contact)** (مستضاف بواسطة المُحافظ على المشروع)؛ يعمل التطبيق الكامل في متصفحك، دون تثبيت.
- **البناء من المصدر** — للمساهمين؛ راجع [README الإنجليزية](../../README.md#build-from-source).

## الميزات

- محرر بصري — انقر أي مفتاح لتعيين مخرجه لأي مجموعة من مفاتيح التعديل وحالة المفتاح الميت
- المفاتيح الميتة، والنهائيات، وخرائط مفاتيح التعديل
- الأشكال الهندسية للوحة المفاتيح المادية ANSI / ISO / JIS
- يقرأ ويكتب صيغ XML الأصلية الخاصة بـ Apple لملفات `.keylayout` وحِزم `.bundle`
- التحقق مع إصلاح تلقائي بنقرة واحدة؛ وتصدير PNG وورقة مرجعية
- واجهة بـ 24 لغة

- **حفظ مقابل حفظ باسم** — الحفظ يستبدل الملف الحالي؛ حفظ باسم ينشئ نسخة.
## الخصوصية

**تطبيق سطح المكتب** لا يجمع أي بيانات ويعمل دون اتصال. **التطبيق المستضاف** على [keymano.ys.contact](https://keymano.ys.contact) يستخدم اختياريًا **Google Analytics** (إحصاءات صفحات مجهولة). عملك على `.keylayout` لا يغادر المتصفح أبدًا. الاستضافة الذاتية بلا تحليلات ما لم يفعّلها المشغّل. راجع [PRIVACY.md](../../PRIVACY.md).
