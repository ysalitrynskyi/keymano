# Keymano — macOS کی بورڈ لےآؤٹ ایڈیٹر

> 🌐 یہ ایک ترجمہ شدہ خلاصہ ہے۔ مکمل دستاویزات انگریزی میں ہیں: **[README](../../README.md)**۔

**Keymano** macOS `.keylayout` فائلوں اور `.bundle` کی بورڈ پیکیجز کے لیے ایک مفت، اوپن سورس، کراس پلیٹ فارم ایڈیٹر ہے — Ukelele کا ایک اوپن سورس متبادل۔ macOS، Windows، Linux، یا براہِ راست اپنے براؤزر میں ایک لائیو، قابلِ کلک کی بورڈ کے ساتھ macOS کی بورڈ لےآؤٹس بنائیں، ترمیم کریں، اور جانچیں۔

## چلانے کے تین طریقے

- **ڈیسک ٹاپ ایپ ڈاؤن لوڈ کریں** — macOS، Windows، یا Linux کے لیے، [Releases page](https://github.com/ysalitrynskyi/keymano/releases) سے۔
- **اسے اپنے براؤزر میں استعمال کریں** — ہوسٹ شدہ ایپ **[keymano.ys.contact](https://keymano.ys.contact)** پر کھولیں (مینٹینر کی میزبانی میں); پوری ایپ آپ کے براؤزر میں چلتی ہے، کسی انسٹالیشن کی ضرورت نہیں۔
- **سورس سے بنائیں** — معاونین کے لیے; [English README](../../README.md#build-from-source) دیکھیں۔

## خصوصیات

- بصری ایڈیٹر — کسی بھی کلید پر کلک کریں تاکہ کسی بھی modifier combination اور dead-key state کے لیے اس کی output مقرر کی جا سکے
- Dead keys، terminators، اور modifier maps
- ANSI / ISO / JIS جسمانی کی بورڈ جیومیٹریز
- Apple's native `.keylayout` XML اور `.bundle` پیکیجز کو پڑھتا اور لکھتا ہے
- ایک کلک auto-repair کے ساتھ validation; PNG اور reference-sheet export
- 24 زبانوں میں انٹرفیس
