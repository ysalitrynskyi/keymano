# Keymano — macOS কীবোর্ড লেআউট সম্পাদক

> 🌐 এটি একটি অনূদিত সারসংক্ষেপ। পূর্ণ ডকুমেন্টেশন ইংরেজিতে আছে: **[README](../../README.md)**।

**Keymano** হলো macOS `.keylayout` ফাইল এবং `.bundle` কীবোর্ড প্যাকেজের জন্য একটি বিনামূল্যের, ওপেন-সোর্স, ক্রস-প্ল্যাটফর্ম সম্পাদক — Ukelele-এর একটি ওপেন-সোর্স বিকল্প। macOS, Windows, Linux, বা সরাসরি আপনার ব্রাউজারে একটি লাইভ, ক্লিকযোগ্য কীবোর্ড ব্যবহার করে macOS কীবোর্ড লেআউট তৈরি, সম্পাদনা, এবং পরিদর্শন করুন।

## চালানোর তিনটি উপায়

- **ডেস্কটপ অ্যাপ ডাউনলোড করুন** — macOS, Windows, বা Linux-এর জন্য, [Releases page](https://github.com/ysalitrynskyi/keymano/releases) থেকে।
- **আপনার ব্রাউজারে ব্যবহার করুন** — হোস্ট করা অ্যাপটি **[keymano.ys.contact](https://keymano.ys.contact)**-এ খুলুন (রক্ষণাবেক্ষণকারী দ্বারা হোস্ট করা); সম্পূর্ণ অ্যাপটি আপনার ব্রাউজারেই চলে, ইনস্টল করার দরকার নেই।
- **সোর্স থেকে বিল্ড করুন** — অবদানকারীদের জন্য; [English README](../../README.md#build-from-source) দেখুন।

## বৈশিষ্ট্য

- ভিজ্যুয়াল সম্পাদক — যেকোনো modifier combination এবং dead-key state-এর জন্য এর আউটপুট সেট করতে যেকোনো কী-তে ক্লিক করুন
- Dead keys, terminators, এবং modifier maps
- ANSI / ISO / JIS ফিজিক্যাল কীবোর্ড জ্যামিতি
- Apple-এর নেটিভ `.keylayout` XML এবং `.bundle` প্যাকেজ পড়ে ও লেখে
- এক-ক্লিক auto-repair সহ validation; PNG এবং reference-sheet export
- 24টি ভাষায় ইন্টারফেস
