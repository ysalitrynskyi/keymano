# Keymano — Editor di layout tastiera per macOS

> 🌐 Questo è un riepilogo tradotto. La documentazione completa è in inglese: **[README](../../README.md)**.

> 🟢 **Nuovo su GitHub o non sei uno sviluppatore?** Leggi la **[guida introduttiva (inglese)](../../docs/GETTING_STARTED.md)** in linguaggio semplice.

**Keymano** è un editor gratuito, open-source e multipiattaforma per i file `.keylayout` di macOS e i pacchetti tastiera `.bundle` — un'alternativa open-source a Ukelele. Crea, modifica e ispeziona layout di tastiera macOS con una tastiera live e cliccabile, su macOS, Windows, Linux o direttamente nel tuo browser.

## Tre modi per eseguirlo

- **Scarica l'app desktop** — per macOS, Windows o Linux, dalla [pagina Releases](https://github.com/ysalitrynskyi/keymano/releases).
- **macOS:** Unsigned build — after installing from the `.dmg`, see **[First launch on macOS (English)](../../docs/GETTING_STARTED.md#first-launch-on-macos-important)** for the one-time Terminal step if macOS blocks the app.
- **Usalo nel tuo browser** — apri l'app ospitata su **[keymano.ys.contact](https://keymano.ys.contact)** (ospitata dal manutentore); l'app completa funziona nel tuo browser, senza installazione.
- **Limiti della versione web:** nel browser puoi aprire file `.keylayout` indipendenti ed esportarli come `.keylayout` o `.bundle.zip`. Il browser non può importare direttamente cartelle `.bundle`, installare in `~/Library/Keyboard Layouts/` o sfogliare layout di sistema installati. Per queste azioni usa l’app desktop.
- **Compila dal sorgente** — per i contributori; consulta il [README in inglese](../../README.md#build-from-source).

## Funzionalità

- Editor visivo — fai clic su qualsiasi tasto per impostarne l'output per qualsiasi combinazione di modificatori e stato di dead key
- Dead keys, terminatori e mappe dei modificatori
- Geometrie fisiche delle tastiere ANSI / ISO / JIS
- Legge e scrive i pacchetti XML nativi `.keylayout` e `.bundle` di Apple
- Validazione con riparazione automatica a un clic; esportazione PNG e fogli di riferimento
- Interfaccia in 24 lingue

- **Salva vs. Salva con nome** — Salva sovrascrive il file corrente; Salva con nome crea una copia.
## Privacy

L’**app desktop** non raccoglie dati e funziona offline. L’**app web ospitata** su [keymano.ys.contact](https://keymano.ys.contact) usa opzionalmente **Google Analytics** (statistiche anonime delle pagine). Il tuo lavoro `.keylayout` non lascia mai il browser. Le istanze self-hosted non hanno analitiche salvo attivazione. Vedi [PRIVACY.md](../../PRIVACY.md).
