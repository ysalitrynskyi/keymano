# Keymano — ویرایشگر چیدمان صفحه‌کلید macOS

> 🌐 این یک خلاصهٔ ترجمه‌شده است. مستندات کامل به زبان انگلیسی است: **[README](../../README.md)**.

**Keymano** یک ویرایشگر رایگان، متن‌باز و چندسکویی برای فایل‌های `.keylayout` و بسته‌های صفحه‌کلید `.bundle` در macOS است — جایگزینی متن‌باز برای Ukelele. چیدمان‌های صفحه‌کلید macOS را با یک صفحه‌کلید زنده و قابل‌کلیک، روی macOS، Windows، Linux، یا مستقیماً در مرورگر خود بسازید، ویرایش کنید و بررسی کنید.

## سه راه برای اجرا

- **دانلود برنامهٔ دسکتاپ** — برای macOS، Windows، یا Linux، از [صفحهٔ Releases](https://github.com/ysalitrynskyi/keymano/releases).
- **استفاده در مرورگر** — برنامهٔ میزبانی‌شده را در **[keymano.ys.contact](https://keymano.ys.contact)** باز کنید (میزبانی‌شده توسط نگهدارنده)؛ کل برنامه در مرورگر شما اجرا می‌شود، بدون نیاز به نصب.
- **ساخت از سورس** — برای مشارکت‌کنندگان؛ به [README انگلیسی](../../README.md#build-from-source) مراجعه کنید.

## ویژگی‌ها

- ویرایشگر بصری — روی هر کلید کلیک کنید تا خروجی آن را برای هر ترکیب از کلیدهای تغییر‌دهنده و وضعیت کلید مرده تنظیم کنید
- کلیدهای مرده، terminatorها، و نگاشت‌های تغییر‌دهنده
- هندسه‌های فیزیکی صفحه‌کلید ANSI / ISO / JIS
- خواندن و نوشتن XML بومی Apple در قالب `.keylayout` و بسته‌های `.bundle`
- اعتبارسنجی با تعمیر خودکارِ یک‌کلیکی؛ خروجی PNG و برگهٔ مرجع
- رابط کاربری در 24 زبان
