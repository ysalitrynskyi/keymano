# Keymano — Éditeur de disposition de clavier pour macOS

> 🌐 Ceci est un résumé traduit. La documentation complète est en anglais : **[README](../../README.md)**.

**Keymano** est un éditeur gratuit, open-source et multiplateforme pour les fichiers `.keylayout` de macOS et les paquets de clavier `.bundle` — une alternative open-source à Ukelele. Créez, modifiez et inspectez des dispositions de clavier macOS avec un clavier interactif cliquable, sur macOS, Windows, Linux, ou directement dans votre navigateur.

## Trois façons de l’exécuter

- **Télécharger l’application de bureau** — pour macOS, Windows ou Linux, depuis la [page des versions](https://github.com/ysalitrynskyi/keymano/releases).
- **L’utiliser dans votre navigateur** — ouvrez l’application hébergée à **[keymano.ys.contact](https://keymano.ys.contact)** (hébergée par le mainteneur) ; l’application complète s’exécute dans votre navigateur, sans installation.
- **Compiler à partir des sources** — pour les contributeurs ; voir le [README en anglais](../../README.md#build-from-source).

## Fonctionnalités

- Éditeur visuel — cliquez sur n’importe quelle touche pour définir sa sortie pour toute combinaison de modificateurs et tout état de touche morte
- Touches mortes, terminateurs et cartes de modificateurs
- Géométries physiques de clavier ANSI / ISO / JIS
- Lit et écrit le XML natif `.keylayout` d’Apple et les paquets `.bundle`
- Validation avec réparation automatique en un clic ; export en PNG et en feuille de référence
- Interface en 24 langues
